    GitHub Search

    This project is about searching different repositories of different people.
    It requires the user to know the GitHub name of the searched person.
    Thes user will get the informations about that person from the GitHub.
    The user will also get to know the time that serached repository has been created.

    HTML
    CSS
    Bootstrap
    Angular JS
