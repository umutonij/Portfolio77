    Gallery

    This project is about photos.
    A user will be able to see different photos.
    A user can be able to click on each photo and that photo will be able to be expanded.
    After that they will be able to get full details about each photo.
    Photos are diplayed based on their categories.
    And finally you can copy the link of the photo and send it to someone.

    Technologies:

    HTML
    CSS
    Django
    django-bootstrap3==11.0.0
    psycopg2==2.7.7
    postgreSQL
